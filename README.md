This is the **preliminary** code for the following paper:

*Lianhui Qin,  Vered Shwartz, Peter West, Chandra Bhagavatula, Jena Hwang, Ronan Le Bras, Antoine Bosselut, Yejin Choi   
EMNLP 2020*
